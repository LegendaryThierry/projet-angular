Mode d'emploi du nouveau système de note de l'Université Côte D'Azur

Comment l'utiliser ?

Comme première étape, munissez-vous de vos identifiants de connexion.

Vous avez deux possibilités:
-Vous pouvez vous connecter à l'aide de votre identifiant et de votre mot de passe
-Vous pouvez vous créer un compte.

C'est fait ? excellent !

Désormais vous arrivez sur la page d'accueil, sur le menu en haut à gauche, vous pouvez voir:

-Votre nom,
-Votre prénom,
-Votre photo,
-Activer le "Dark Mode"

Si vous êtes un élève:
Vous allez pouvoir observer:
-vos devoirs non rendu,
-vos devoirs rendu

Si vous êtes un professeur
Vous pourrez en plus,
-voir les données des devoirs,
-en donner.
-vous pouvez, en cliquant sur un des devoirs,
-le noter (ou le supprimer si vous voulez que l'étudiant ait 0)
